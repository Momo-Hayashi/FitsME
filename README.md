# README
  
## FitsME


## 開発言語
- Ruby 2.6.5  
- Ruby on Rails 5.2.5  

## 就業Termの技術
- devise  
- お気に入り機能  
- kaminari  
- Ajax  

## カリキュラム外の技術
- Pay.jp  
- ancestry  
- ActiveStorage  

## 余裕があれば実装
- AWS EC2  
- ransack  
- omniauth  

## 実行手順
```
$ git clone git@github.com:Momo-Hayashi/FitsME.git  
$ cd FitsME  
$ bundle install  
$ rails db:create db:migrate  
$ rails s  
```
