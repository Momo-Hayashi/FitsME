

# README
  
## FitsME


## 開発言語
- Ruby 2.6.5  
- Ruby on Rails 5.2.5  

## 就業Termの技術
- devise  
- お気に入り機能 　

## カリキュラム外の技術
- Pay.jp  
- ancestry  
- ActiveStorage  

## その他の技術（時間に猶予があれば実行）
- AWS EC2  
- kaminari  
- Vue.js  
- ransack  
- omniauth  

## 実行手順
```
$ git clone git@github.com:Momo-Hayashi/FitsME.git  
$ cd FitsME  
$ bundle install  
$ rails db:create db:migrate  
$ rails s  
```
